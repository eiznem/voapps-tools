First Time Setup

Thank you for downloading VoApps Tools! Follow these simple steps to get started.

Installation Steps

1 Drag to Applications

Drag the VoApps Tools icon to your Applications folder.

2 Remove Security Block

Choose one of these methods:

Method A: Run the Script Easiest
  1. Double-click "Remove Quarantine.command"
  2. If you see "cannot be opened because it is from an unidentified developer":
    • Right-click on the file instead
    • Select "Open"
    • Click "Open" in the dialog
  3. The script will automatically allow the app to run
Method B: Right-Click to Open Simple
  1. Open your Applications folder
  2. Find "VoApps Tools"
  3. Right-click (or Control-click) on the app
  4. Select "Open" from the menu
  5. Click "Open" in the warning dialog
